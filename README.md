## 项目介绍

作为本人本科毕设***GPU监测系统***前端项目，使用vue的echarts来进行数据可视化展示，实现了用户注册登录的index与布局的layout，由于同时进行前后端开发，精力有限，仍在陆续修进。

<br><br><br><br>

## 如何运行(需要配合前端一起启动以及golang环境)
**部署项目**
```
git clone https://github.com/dianayyds/GPU-backend.git
```

**进入项目**
```
cd GPU-backend
```

**下载依赖**
```
go mod tidy
```

**启动**
```
go run main.go
```
